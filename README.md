# Blogger Backend API 🖥️

The robust backend RESTful API for a full-stack blogging application. This project was developed as a practical environment to master server-side Node.js development, secure authentication flows, and API design.

## 🎯 Learning & Practice Goals
* **Secure Authentication:** Implementing robust User Authentication using JSON Web Tokens (JWT).
* **State/Session Handling:** Managing user sessions safely using HTTP-only cookies to prevent XSS attacks.
* **API Architecture:** Designing clean, predictable REST endpoints using the MVC (Model-View-Controller) pattern.

## 🛠️ Tech Stack
* **Runtime Environment:** Node.js
* **Framework:** Express.js
* **Authentication:** JSON Web Tokens (JWT) & `cookie-parser`
* **Hosting:** Render
